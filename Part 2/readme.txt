Instructions: 

1. follow github link and download files by selecting the green "Code" tab, select "download zip"
2. create a folder called "java" inside your C: drive
3. open folder and select "extract all"
4. type "C:\java" into input area for where files will be extracted to
5. download java JDK from this link: https://www.oracle.com/java/technologies/downloads/#jdk17-windows
6. Download and Install "x64 Installer" from the website
7. open command prompt by typing CMD in the start menu 
8. go to java directory by typing command: cd\java\CS-II-Group-Project-main\Part 2\Main Program
8. Compile the program with the command: javac MainMenu.java
9. Run the program with the command: java MainMenu.java
