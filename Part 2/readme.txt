Instructions: 

1. download files and unzip them
2. create a folder called "java" inside your C: drive
3. copy java files to C:\java
4. download java JDK from this link: https://www.oracle.com/java/technologies/downloads/#jdk17-windows
5. Download and Install "x64 Installer" from the website
6. open command prompt by typing CMD in the start menu 
7. go to java directory by typing command: cd java
8. Compile the program with the command: javac MainMenu.java
9. Run the program with the command: java MainMenu.java
